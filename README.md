# _Template_

## Настройка проекта:

1.  Открыть редактор кода от имени администратора;
2.  Установить необходимые пакеты командой `npm i`;

## Работа с проектом:

- `npm run dev` - режим разработчика;
- `npm run build` - режим продакшн;
